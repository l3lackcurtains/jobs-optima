# Security Policy

## Supported Versions

Only the latest release on `main` receives security fixes.

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Report privately via [GitHub Security Advisories](https://github.com/l3lackcurtains/resume-builder/security/advisories/new) — this keeps the disclosure private until a fix is ready.

Include:
- Description of the vulnerability and its potential impact
- Steps to reproduce
- Suggested fix if you have one

You can expect acknowledgement within 48 hours and a resolution timeline within 7 days for critical issues.

## Scope

Jobs Optima handles user credentials (JWT tokens, hashed passwords) and AI API keys. Reports related to authentication bypass, credential exposure, or injection attacks are highest priority.
